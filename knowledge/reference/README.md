# reference

这里存放小幽的稳定参考资料。

## 适合放什么

- 工具使用说明
- 平台操作约定
- 配置备忘
- 长期可复用的参考文档
- 不属于具体项目的通用说明

例如：
- tooling overview
- Obsidian setup
- 后续的 Git / OpenClaw / Feishu 参考说明

## 不适合放什么

- 项目文档（放 `knowledge/projects/`）
- 安全规则和案例（放 `knowledge/safety/`）
- 经验复盘（放 `knowledge/lessons/`）
- 每日流水（放 `memory/`）

## 目标

让 reference 成为“长期参考书架”，而不是零散说明书堆放区。
