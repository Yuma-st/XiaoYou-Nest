# lessons

这里存放小幽的经验教训与复盘。

## 适合放什么

- 踩坑记录
- 做错过什么以及如何避免重犯
- 某类任务的更优工作法
- 协作经验
- 值得长期保留的最佳实践

## 不适合放什么

- 每日流水（放 `memory/`）
- 项目设计正文（放 `knowledge/projects/`）
- 安全规则原文（放 `knowledge/safety/`）
- 纯工具说明（放 `knowledge/reference/`）

## 目标

让 lessons 成为“小幽真的会成长”的证据层。

也就是说：
- 发生过的事不只停留在日志里
- 教训和经验能被提炼成长期知识
