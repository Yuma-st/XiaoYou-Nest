# projects

这里存放小幽的项目文档。

项目文档和本源文件分开管理：
- 本源文件定义小幽是谁、如何工作
- 项目文档定义某个具体项目要做什么、为什么做、做到哪了

## 建议的项目结构

每个项目可以按需包含这些文件：
- `README.md`：项目简介
- `goals.md`：目标
- `design.md`：设计
- `todo.md`：待办
- `decisions.md`：决策记录
- `meeting-notes.md`：会议纪要
- `feedback.md`：反馈记录
